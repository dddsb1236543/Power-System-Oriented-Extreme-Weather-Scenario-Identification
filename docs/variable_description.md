\# Variable Description



This document summarizes the types of variables used in the case study.



Due to confidentiality restrictions, only variable categories and distribution-level descriptions are provided. Raw values, original timestamps, equipment-level information, and identifiable grid-operation records are not released.



\## Meteorological Variables



| Variable type | Description |

|---|---|

| Temperature | Used to characterize high-temperature and low-temperature abnormal conditions |

| Wind speed | Used to characterize wind-related disturbances and typhoon-related conditions |

| Solar irradiance | Used to characterize solar-resource variation and renewable-output uncertainty |

| Precipitation-related variable | Used to characterize heavy-precipitation or rainy-season-related abnormal conditions, if available |



\## Power-System Variables



| Variable type | Description |

|---|---|

| Load | Used to characterize demand-side response under extreme weather |

| Wind power | Used to calculate renewable-output variation and net-load pressure, if available |

| Solar power | Used to calculate renewable-output variation and net-load pressure, if available |

| Net load | Derived indicator for source-load balance analysis |

| Net-load ramp | Derived indicator for operational flexibility and reserve requirement analysis |

| Reserve requirement | Derived indicator for power-system risk interpretation |



\## Derived Risk Indicators



The power-system-oriented interpretation is supported by aggregated risk indicators, including:



\- peak load;

\- net load;

\- net-load ramp;

\- renewable-output variation;

\- reserve requirement;

\- source-load imbalance indicator.



These indicators are used only at the aggregated or distribution level in the public repository.
